# RASP

## Overview
RASP is auto build and application manager.

## Install

build go module and set the env.

```
% git clone https://github.com/masato-ka/rasp
% cd rasp
% go build
% export PATH=$PATH:./rasp
```

## Usage

### build

 * for Python and Go project.

    ```
    % rasp build -repo=<git repository> -dst=<>
    ```
